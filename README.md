Landing-Page, Modul-Seiten und Rechtstexte der App **ClassTiles**, gehostet über GitHub Pages.

**Deutsch (maßgeblich):** `index.html`, `modul-*.html` (6), `datenschutz.html`, `impressum.html`, `nutzungsbedingungen.html`, `support.html`

**English (convenience translation):** `en/index.html`, `en/module-*.html` (6), `en/privacy.html`, `en/imprint.html`, `en/terms.html`, `en/support.html`

`assets/` enthält Video, Standbild und Screenshots (sprachneutral).

`docs/legal/` (+ `docs/legal/en/`), `scripts/legal-pages/build.py`, `modules.py`, `faq*.py`.
